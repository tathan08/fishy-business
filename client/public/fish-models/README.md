# Fish Model Icons

Place your fish model icon images in this directory with the following exact filenames:

- `swordfish.png` - Icon for Swordfish
- `blobfish.png` - Icon for Blobfish
- `pufferfish.png` - Icon for Pufferfish
- `shark.png` - Icon for Shark
- `sacabambaspis.png` - Icon for Sacabambaspis

## Image Requirements

- **Format**: PNG (recommended) or JPG
- **Size**: 128x128 pixels or larger (will be scaled down)
- **Transparent Background**: Recommended for PNG files
- **Style**: Should match the ocean/fish theme of the game

## Usage

These icons will be displayed on the join page when players select their fish species before entering the game.

Once you add the images, the game will automatically load them instead of showing the emoji placeholders.
